# HAINOA Site

Version: v0.1  
Status: Active architecture site

This repository contains the public website for **HAINOA**.

**HAINOA** stands for **Hybrid AI Native Open Architecture**. It is the open architecture layer of the HAIN framework and provides the deeper structural and interpretive explanation beneath the public-facing HAIN World layer.

## Role in the HAIN system

Within the current HAIN structure:

- **HAIN World** introduces
- **HAINOA** explains
- **HAINIZE** enables practical entry

This repository belongs to the second layer: the architecture core.

## What this repository contains

This repository currently contains:

- the architecture-facing public site for HAINOA
- explanatory content around the Hybrid AI Native framework
- links to the whitepaper, glossary, and HAIN World
- site assets such as favicon and OG image
- the domain configuration for `hainoa.org`

## Where to start

Start with:

- `index.html` — the main architecture site homepage
- `CNAME` — domain binding for `hainoa.org`
